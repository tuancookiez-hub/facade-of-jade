# Facade of Jade

An interactive Wuxia drama inspired by the 2005 cult classic *Façade*. The AI is
the load-bearing creative core: a wandering swordsman responds in real time to
your choices, and the story's emotional state shifts as you talk. No
branching-script illusion — every line is generated by a small open model
(Qwen3-4B-Instruct) running through `llama.cpp` on Modal.

## How it works

- **Frontend** — Gradio `ChatInterface` with custom Wuxia CSS theming
- **Inference** — `llama-cpp-python` on Modal (A10G, Q4_K_M GGUF)
- **Drama manager** — `beats.py` tracks mood, trust, discourse acts, and story beats
- **Dynamic prompts** — each player line changes the system prompt sent to Modal
- **No hosted model API** — the model runs through `llama.cpp` on our Modal backend
